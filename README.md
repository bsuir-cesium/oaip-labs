# oaip-labs

Лабораторные работы по дисциплине «Основы алгоритмизации и программирования» (ОАиП).
Язык реализации: Object Pascal (Delphi / Free Pascal), частично Python.

## Содержание

### [`lab-4`](lab-4) — Одномерные массивы

**Тема:** «Одномерные массивы»

Дана последовательность символов. Найти символ, который встречается минимум дважды, причём между любыми двумя его вхождениями есть хотя бы один символ из набора {A, B}. Реализовано на двух языках (Pascal и Python) с набором unit-тестов.

> Вариант задания из отчёта — условие было вставлено изображением, восстановлено по коду.

### [`lab-3`](lab-3) — Итерационные циклы

**Тема:** «Итерационные циклы» | **Вариант:** 30

Сравнение прямого вычисления функции `f(x) = ((1-x)/x) · ln(1/(1-x))` с приближением через ряд Тейлора. Программа демонстрирует сходимость ряда для значений x от −0.6 до 0.95 при точностях ε = 10⁻², 10⁻³, 10⁻⁴, подсчитывая необходимое количество членов ряда. Результат выводится в виде таблицы.

### [`lab-5`](lab-5) — Сортировки

**Тема:** «Сортировка массивов»

Сравнение трёх алгоритмов: улучшенный пузырёк (с границей), шейкерная сортировка и QuickSort. Тестирование на массивах размером 10, 100 и 2000 элементов в трёх состояниях: случайный, отсортированный и обратный порядок. Для каждого алгоритма подсчитываются количество сравнений и перестановок.

### [`lab-6`](lab-6) — Двумерные массивы

**Тема:** «Массивы» (двумерные)

Определение «похожих» строк в матрице 6×6: две строки считаются похожими, если содержат одинаковый набор уникальных элементов (независимо от порядка и повторений). Используются вспомогательные подпрограммы `InArray`, `SortArray`, `AreRowsSimilar`.

### [`lab-7`](lab-7) — Строки

**Тема:** «Строки»

Две части. **Часть 1** (английский текст): поиск слов, совпадающих с суффиксами алфавита, и удаление гласных из всех слов кроме последнего. **Часть 2** (русский текст): валидация предложений по системе правил (заглавная первая буква, корректная пунктуация, отсутствие лишних пробелов, все слова содержатся в словаре ~1.5 млн слов), замена недопустимых символов звёздочками и выравнивание текста по ширине k символов.

> Реализация на Python и Delphi. Включены unit-тесты. Правила валидации взяты из отчёта.

### [`lab-9`](lab-9) — Алгоритмы поиска

**Тема:** «Поиск в массиве»

Реализация и сравнение бинарного поиска и блочного (квадратичного) поиска в отсортированном массиве из 1000 записей (число + имя сенсора). Поддерживается поиск по имени, по значению и по диапазону. Для каждого алгоритма подсчитывается количество обращений к массиву.

### [`lab-2-3`](lab-2-3) — Рекурсия

**Тема:** «Рекурсия»

Две классические задачи. Первая — решение головоломки **Ханойские башни** с n дисками. Вторая — генерация всех перестановок массива целых чисел методом backtracking.

> Написано на Free Pascal. Сборка через Makefile.

### [`lab-recurrent`](lab-recurrent) — Рекурсия (доп.)

**Тема:** «Рекурсия»

Вычисление произведения всех отрицательных элементов массива с помощью рекурсивной функции. Массив генерируется случайно в диапазоне [−50, 49].

> Без git-истории (локальный проект, ~апрель 2025).

### [`lab-2-7`](lab-2-7) — Множества и синтаксический анализ

**Тема:** «Множества»

Два задания на работу с `Set of Char`. Первое — поиск повторяющихся гласных букв в словах текста. Второе — рекурсивный парсер для валидации математических формул вида `(Формула Знак Формула)` с поддержкой операций `+`, `-`, `*`.

> Без git-истории (локальный проект, ~май 2025).
