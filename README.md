# oaip-labs

Лабораторные работы по дисциплине «Основы алгоритмизации и программирования» (ОАиП).
Язык реализации: Object Pascal (Delphi / Free Pascal), частично Python.

## Содержание

### lab-2-3 — Рекурсия: Ханойские башни и перестановки

Две классические задачи на рекурсию. Первая — решение головоломки **Ханойские башни** с n дисками. Вторая — генерация всех перестановок массива целых чисел методом backtracking.

> Написано на Free Pascal. Сборка через Makefile.

### lab-2-7 — Множества символов и синтаксический анализ

Два задания на работу с `Set of Char`. Первое — поиск повторяющихся гласных букв в словах текста. Второе — рекурсивный парсер для валидации математических формул вида `(Формула Знак Формула)` с поддержкой операций `+`, `-`, `*`.

> Без git-истории (локальный проект).

### lab-3 — Вычисление функции рядом Тейлора

Сравнение прямого вычисления функции `f(x) = ((1-x)/x) * ln(1/(1-x))` с приближением через ряд Тейлора. Программа демонстрирует сходимость ряда для различных значений x и точностей (ε = 10⁻², 10⁻³, 10⁻⁴), подсчитывая необходимое количество членов ряда.

### lab-4 — Анализ последовательностей символов

Поиск символа, который встречается минимум дважды в последовательности, причём между любыми двумя его вхождениями есть хотя бы один символ из набора {A, B}. Реализовано на двух языках (Pascal и Python) с набором тестов.

### lab-5 — Сравнительный анализ алгоритмов сортировки

Сравнение трёх алгоритмов: улучшенный пузырёк (с границей), шейкерная сортировка и QuickSort. Тестирование на массивах размером 10, 100 и 2000 элементов в трёх состояниях: случайный, отсортированный и обратный порядок. Подсчитываются количество сравнений и перестановок.

### lab-6 — Поиск похожих строк в матрице

Определение «похожих» строк в матрице 6×6: две строки считаются похожими, если содержат одинаковый набор уникальных элементов (независимо от порядка и повторений). Используется пузырьковая сортировка для сравнения наборов.

### lab-7 — Обработка текстовых строк

Две части. Часть 1 (английский текст): поиск слов, совпадающих с суффиксами алфавита, и удаление гласных из всех слов кроме последнего. Часть 2 (русский текст): валидация предложений с использованием словаря из ~1.5 млн слов (бинарный поиск), замена недопустимых символов и выравнивание текста по ширине.

> Реализация на Python и Delphi. Включены unit-тесты.

### lab-9 — Алгоритмы поиска в массиве

Реализация и сравнение бинарного поиска и блочного (квадратичного) поиска в отсортированном массиве из 1000 структур (число + имя сенсора). Поддерживается поиск по имени, по значению и по диапазону. Для каждого алгоритма подсчитывается количество обращений к массиву.

### lab-recurrent — Рекурсивное произведение отрицательных чисел

Вычисление произведения всех отрицательных элементов массива с помощью рекурсивной функции. Массив генерируется случайно в диапазоне [-50, 49].

> Без git-истории (локальный проект).
